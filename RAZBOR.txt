1. heroui - библиотека компонентов.
2. config - конафигурационная папка, в ней храним повторяющиеся данные, вычисляемые значния и т.п. Внутри создаем сколько угодно файлов, название должно состоять из name.config.ts. Name - это уникальное название по теме данных, например, если это данные для layout, то layout.config.ts, если общие данные сайта, то site.config.ts. Внутри файлов создаем и экспортируем объект, в который записываем нужные значения.
3. prisma orm - современный ORM (Object-Relational Mapping) для работы с базами данных. Упрощает взаимодействие с БД (PostgreSQL, MySQL, SQLite, MongoDB, SQL Server и др.), предоставляя удобный и безопасный API.
4. PostgreSQL (или просто Postgres) — это сервер базы данных (СУБД), который хранит, управляет и обрабатывает данные. В этом проекте использовался как локальная база данных.
5. pgAdmin - графическая программа для управления базами, это просто удобный интерфейс для работы с БД в PostgreSQL.
